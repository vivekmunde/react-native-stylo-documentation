Alpha colors

Alpha colors are an extension of rgb() color values with an alpha value i.e. opacity. Its RGB value is rgb(255, 255, 255) & the alpha value ranges from 0.1 to 1.

For demonstration purpose, the Alpha color boxes are displayed on different background colors.

Alpha1rgba(255, 255 ,255, 0.1)
Alpha2rgba(255, 255 ,255, 0.2)
Alpha3rgba(255, 255 ,255, 0.3)
Alpha4rgba(255, 255 ,255, 0.4)
Alpha5rgba(255, 255 ,255, 0.5)
Alpha6rgba(255, 255 ,255, 0.6)
Alpha7rgba(255, 255 ,255, 0.7)
Alpha8rgba(255, 255 ,255, 0.8)
Alpha9rgba(255, 255 ,255, 0.9)
Alpha10rgba(255, 255 ,255, 1)

Inverted Alpha colors

Inverted Alpha colors are an extension of rgb() color values with an alpha value i.e. opacity. Its RGB value is rgb(0, 0, 0) & the alpha value ranges from 0.1 to 1.

For demonstration purpose, the Inverted Alpha color boxes are displayed on different background colors.
