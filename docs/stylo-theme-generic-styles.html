Generic styles

Generic styles are the common style definitions like font colors, border, flex or padding. These styles are applicable for multiple React Native components. For example, border styles can be applied to View, ScrollView or TouchableOpacity, font color styles can be applied to Text or TextInput. So the theme has categorized them as generic styles & has defined them separately for better reusability.
