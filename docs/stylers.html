Stylers

Stylers are the styling hooks. Stylers accept StyleNames & optional StyleNamespace as its arguments & return the style object. They read the styles defined for the StyleNames under the StyleNamespace in the theme, combines these styles into one & returns the final style, a standard React Native style object.

Stylo's objective is to provide a styling hook for each React Native component. This way it plans to cover the styling requirements of all the components.

Each React Native component has its own style type. E.g. Text component needs style object of type StyleProp<TextStyle>, View component needs style object of type StyleProp<ViewStyle> etc. So each styler is tightly coupled with a React Native component which returns a style object specific to that component.

Styler arguments

All stylers accept two arguments.

styleNames (Required)

The StyleNames which define the styles.

styleNamespace (Optional)

The Namespace to be used which holds the style definitions for the StyleNames supplied to the hook.
If not provided then the styler hook uses a matching default namespace from the collection of it's Default Namespaces.

useImageBackgroundStyles()

Type definition

function useImageBackgroundStyles<TStyleName extends string, TStyleNamespace extends string>( styleNames: TStyleName[], styleNamespace?: TStyleNamespace, ): StyleProp<ViewStyle> { return useStyles<ViewStyle, T>({ styleNamespace: styleNamespace ?? 'ImageBackgroundStyles', styleNames, }); }

Usage

import React, { useRef } from 'react'; import { ImageBackground, StyleSheet } from 'react-native'; import { TImageBackgroundStyle } from '../themes/types'; const ComponentA = () => { const styles = useRef( StyleSheet.create({ imageBackgroundStyle: useImageBackgroundStyles<TImageBackgroundStyle>(['BackgroundColor.Grey2']), }) ).current; return ( <ImageBackground style={styles.imageBackgroundStyle} {...otherProps} /> ); }
