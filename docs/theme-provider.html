<ThemeProvider />

As the name says, the ThemeProvider is a component responsible for providing the theme values across the application. The ThemeProvider creates a React Context to hold the theme values & ideally sits at the root of the application layout.

In the main file of the application, wrap the application inside the ThemeProvider & supply the theme values to it.

import { ThemeProvider } from 'react-native-stylo'; import { variables, styles } from './themes/default'; const App = () => ( <ThemeProvider variables={variables} styles={styles}> // Application layout </ThemeProvider> );

Props

variables (Required)

The variables defined in the theme. These variables can be accessed across the application using the hook useVariables().

styles (Required)

The style definitions defined in the theme. These styles are used by the Stylish components & Styler hooks to apply them to components.
